# Nine Sols Mods

[BepInEx] plugins for Nine Sols, a Unity game.

## Build

Plugins depend on a NuGet package [NineSols.GameLibs] that is on a custom feed,
so you'll have to add the source for it before being able to build.


[NineSols.GameLibs]: https://github.com/nozwock/NineSols.GameLibs
[BepInEx]: https://github.com/BepInEx/BepInEx
